# CONTRIBUTING

- Fork repository ini.
- Clone repository yang sudah kalian fork tersebut, caranya:
    ```
    git clone https://github.com/USERNAME/SunQA
    ```
- Lakukan perubahan / penambahan / penghapusan pada repository.
- Commit perubahaan.
- Kemudian baru di Push.
- Buat Pull Request.
- Happy waiting :dancer:.
